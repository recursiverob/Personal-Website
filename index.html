ScoreMyStocks | Stock/Portfolio Anaylzer

My implementation of a stock/sportfolio analysis tool that evaluates stocks using several financial metrics such as P/E Ratio, PEG Ratio, ROE, Profit Margin, Revenue Growth, Earnings Growth, Beta, Dividend Yield, and 52-Week High/Low. The tool integrates with the Alpha Vantage API to fetch real-time data, categorizes stocks into performance tiers (Very Good, Good, Neutral, Poor), and provides recommendations with data visualizations.

Tech Stack Used: React | JavaScript | Vite (just for setting and bundling up the dev environment) | Alpha Vantage API

MenuScan | AI Menu Scanner

A small MVP app I developed using React Native to streamline ordering at restaurants. It captures an image of a menu using a phone's camera, and then extracts dish names, prices, and descriptions using OCR (Google Vision API & Tesseract.js), and displays relevant dish images with brief yet appealing AI-generated descriptions (GPT-3.5-turbo).

Tech Stack Used: React Native | TypeScript | Expo | Google Vision API | Tesseract.js | Google Custom Search API | OpenAI GPT API

SpoutScout | Drinking Fountain Locater

A website that helps users locate accessible drinking fountains across the United States, with over 4,500 fountains mapped. Tech Stack Used: JavaScript | Python | HTML/CSS | JSON | Google Maps API | Nominatim API
